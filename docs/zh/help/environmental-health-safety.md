# Ultralytics 环境、健康与安全 (EHS) 政策

在 Ultralytics，我们深知公司的长期成功不仅取决于我们提供的产品和服务，还取决于我们开展业务的方式。我们致力于确保员工、利益相关方和环境的安全与福祉，并将持续努力减轻对环境的影响，同时促进健康与安全。

## 政策原则

1. **合规**：我们将遵守所有与 EHS 相关的适用法律、法规和标准，并尽可能超越这些标准。

2. **预防**：我们将通过实施[风险管理措施](https://docs.ultralytics.com/help/security)并确保所有运营和流程的安全性，努力预防事故、伤害和环境危害。

3. **持续改进**：我们将通过设定可衡量的目标、监控绩效、审计运营情况以及根据需要修订政策和流程，持续改进 EHS 表现。

4. **沟通**：我们将公开沟通 EHS 绩效，并与利益相关方互动，以了解并回应他们的关切和期望。

5. **教育与培训**：我们将对员工和承包商进行适当的 EHS 流程和实践方面的教育和培训。

## 实施措施

1. **责任与问责**：在 Ultralytics 工作或与 Ultralytics 合作的每位员工和承包商都有责任遵守本政策。经理和主管有责任确保本政策在其管辖范围内得到落实。

2. **风险管理**：我们将识别、评估和管理与运营和活动相关的 EHS 风险，以预防事故、伤害和环境危害。

3. **资源分配**：我们将分配必要的资源以确保 EHS 政策的有效实施，包括必要的设备、人员和培训。

4. **应急准备与响应**：我们将制定、维护并测试应急准备和响应计划，以确保能够有效应对 EHS 事件。

5. **监控与审查**：我们将定期监控和审查 EHS 绩效，以发现改进机会并确保达成目标。

本政策体现了我们对[最大限度减少环境足迹](https://www.ultralytics.com/blog/greener-future-through-vision-ai-and-ultralytics-yolo)、确保员工安全与福祉以及持续改进绩效的承诺。

请记住，有效的 EHS 政策的实施需要每一位在 Ultralytics 工作或与 Ultralytics 合作的人员的参与和承诺。我们鼓励您对自己的安全和他人的安全承担个人责任，并爱护我们生活和工作的环境。

## 常见问题

### Ultralytics 的环境、健康与安全 (EHS) 政策是什么？

Ultralytics 的环境、健康与安全 (EHS) 政策是一个全面的框架，旨在确保员工、利益相关方和环境的安全与福祉。它强调遵守相关法律、通过风险管理预防事故、通过可衡量目标实现持续改进、开放沟通以及员工教育和培训。通过遵循这些原则，Ultralytics 致力于最大限度减少环境足迹并推广可持续实践。[详细了解 Ultralytics 对 EHS 的承诺](https://www.ultralytics.com/about)。

### Ultralytics 如何确保遵守 EHS 法规？

Ultralytics 通过遵守所有适用的法律、法规和标准来确保 EHS 法规的合规性。公司不仅努力满足这些要求，还经常通过实施严格的内部政策来超越这些要求。定期进行审计、监控和审查以确保持续合规。经理和主管也有责任确保这些标准在其管辖范围内得到维护。更多详情，请参阅文档页面中的[政策原则部分](#policy-principles)。

### 为什么持续改进是 Ultralytics EHS 政策的关键原则？

持续改进在 Ultralytics 的 EHS 政策中至关重要，因为它确保公司在环境、健康和安全领域持续提升绩效。通过设定可衡量的目标、监控绩效以及根据需要修订政策和流程，Ultralytics 能够适应新的挑战并优化其流程。这种方法不仅能降低风险，还展示了 Ultralytics 对可持续性和卓越的承诺。有关持续改进的实际示例，请查看[实施措施部分](#implementation-measures)。

### 员工在实施 Ultralytics EHS 政策中的角色和职责是什么？

Ultralytics 的每位员工和承包商都有责任遵守 EHS 政策。这包括遵循安全协议、参加必要的培训，以及对自己的安全和他人的安全承担个人责任。经理和主管还有额外的责任，即确保 EHS 政策在其管辖范围内得到有效实施，这涉及风险评估和资源分配。有关责任与问责的更多信息，请参阅[实施措施部分](#implementation-measures)。

### Ultralytics 如何在 EHS 政策中处理应急准备与响应？

Ultralytics 通过制定、维护并定期测试应急计划来处理应急准备与响应，以有效应对潜在的 EHS 事件。这些计划确保公司能够快速高效地响应，最大限度减少对员工、环境和财产的损害。定期进行培训和演练，以确保响应团队为各种紧急情况做好准备。更多背景信息，请参阅[应急准备与响应措施](#implementation-measures)。

### Ultralytics 如何就 EHS 绩效与利益相关方进行沟通？

Ultralytics 通过分享相关信息并回应任何关切或期望，与利益相关方公开沟通 EHS 绩效。这种互动包括定期报告 EHS 活动、绩效指标和改进计划。同时鼓励利益相关方提供反馈，这有助于 Ultralytics 持续完善其政策和实践。在[沟通原则部分](#policy-principles)了解更多关于这一承诺的信息。
